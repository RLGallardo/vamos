RGB e CMYK: Entenda as Diferenças

O Que é RGB?

O Modelo RGB (Red, Green, Blue) é utilizado em telas digitais, onde as cores são formadas pela combinação de luz.

Funcionamento do Pixel no RGB

Cada pixel em uma tela é composto por três subpixels: vermelho, verde e azul. A variação da intensidade de cada um gera diferentes cores.

O que é CMYK?

O modelo CMYK (Cyan, Magenta, Yellow, Black) é usado para impressão, onde as cores são formadas pela absorção de luz.

Comparação entre RGB e CMYK

RGB é utilizado para telas, enquanto CMYK é usado para impressões. A conversão entre os dois pode causar variações nas cores.
